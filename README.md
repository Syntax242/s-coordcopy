# s-coordcopy

A lightweight and developer-friendly FiveM tool that allows you to quickly copy **vector2**, **vector3**, **vector4**, and **heading** coordinates using customizable keybinds.  
Supports both **ox_lib** and **QBCore** notify systems and includes flexible formatting options.
